# VolumeX

A simple, privacy-focused volume control shortcut for Android via Quick Settings.

## Features

- **Quick Access:** Control your system volume directly from the Quick Settings panel.
- **Privacy Focused:** Works entirely offline. No internet permissions, no tracking, no telemetry.
- **Lightweight:** Tiny app footprint with zero background battery drain.
- **Open Source:** Licensed under Apache 2.0.

## F-Droid Checklist

- [x] **Fully Open Source:** Licensed under Apache 2.0.
- [x] **Privacy:** No internet permission, no tracking, no non-free dependencies.
- [x] **Standard Build System:** Uses standard Gradle build.

## Build Instructions

1. Clone the repository.
2. Open in Android Studio.
3. Run `./gradlew assembleDebug` to build the debug APK.
4. Run `./gradlew assembleRelease` to build the production APK.
